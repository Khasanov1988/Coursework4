Программа для парсинга и дальнейшей работы с вакансиями с ресурсов HeadHunter и SuperJob

Для работы необходимо указать в меню источник, ключевое слово и количество страниц выдачи, после чего начнется процедура парсинга.
После удачного парсинга откроется меню, в котором можно работать с данными. Программа поддерживает следующий функционал:
- Сортировка вакансии по убыванию минимальной зарплаты
- Сортировка вакансии по убыванию максимальной зарплаты
- Фильтрация N верхних вакансий списка
- Фильтрация вакансию по ключевому слову
- Фильтрация вакансии в диапазоне зарплат
- Возврат к исходному списку вакансий
"print" - Вывод вакансий в консоль
"exit" - Выход
